Track your workouts using this project!
Using OpenAI's GPT-3 model for natural language processing ,a input from the user saying "I ran for 3 kms."
is taken and enters the "date","time","exercise","duration",and "calories" into your google sheet.
Sheety is used to send get and post requests to read and update data in google sheet.
NutritionixAPI is used to estimate time and calories of exercises entered by the user.mh
